HTML表单和CSS样式

一 HTML表单（重点）

作用：将用户在页面输入的信息（数据），发送给java服务器（php、.net）

<!--
    表单（容器）
    <form></form>
        常用属性：
            action：将表单数据发送给指定的服务器 (url地址)  # 代表当前页面
            name：当前表单的名称
            method：当前表单的提交方式，共有7种，常见2种：get（默认）、post
                get：提交的参数在url地址栏后面进行拼接   ***.html?username=鱼儿水中游&password=12345&birthday=&gender=0&hobby=somke&hobby=perm#
                post：提交的参数不会再url地址蓝后面进行拼接

                区别：
                    1）get提交在地址后进行拼接 相对来说不安全、post提交不会再地址后进行拼接 相对来说按钮
                    2）get提交参数有大小限制（4KB）、post提交参数没有大小的限制


     文本框
     <input/>
        常用属性：
            type：type属性的值不同，展示的样式和功能也不同
                常见的有11个值（必须掌握）
                    text：普通文本框（默认值）
                    password：密码框 特点：输入的都是掩码
                    date：日期选择器 (h5提供的)
                    radio：单选框 特点：name属性相同分为一组，完成互斥的效果
                        checked="checked" | true 当前表单项默认被选中
                    checkbox：复选框 特点：name属性相同分为一组
                        checked="checked" | true 当前表单项默认被选中

                     button：普通按钮（后期学习js，按钮会结合js事件完成某些动作效果）
                     submit：提交按钮 特点：点击提交按钮，触发表单提交 执行action发送请求
                     reset：重置按钮 特点：使表单项恢复到默认状态
                     image：图片提交 特点：与submit功能一致

                     file：文件上传 （必须使用post提交方式）
                     hidden：隐藏域 特点：页面无法看到，用户无法修改或填写 （存放id）
            name：当前表单项的名称
                设置了name属性的表单项才会被提交
                格式：key=value
                ?username=婷哥
                &password=12345
                &birthday=&
                gender=0
                &hobby=somke
                &hobby=perm#
            value：当前表单项的值（让用户去输入）
     下拉列表
     <select name="" >
        <option value="">展示的内容</option>
     </select>
         注意：select标签的name=option标签的value
         selected="selected" true
     文本域
     <textarea name="">我是value</textarea>
        常用属性：
            cols：列宽 默认30
            rows：行高 默认10

-->

二 CSS样式（整体感知）

2.1 概述

层叠样式表（Cascading Style Sheet）

专门用于网页的美化

初体验

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>01_初体验</title>
    <style>
        font{
            color: red;
            font-size: 45px;
            font-weight: bold;
            font-style: italic;
        }
    </style>
</head>
<body>
<!--<font size="7" color="red"><b><i>脚踏实地行</i></b></font> <br>
<font size="7" color="red"><b><i>海阔天空飞</i></b></font> <br>-->

<font>脚踏实地行</font> <br>
<font>海阔天空飞</font> <br>

</body>
</html>

美化优点

实现了样式和内容的分离，提高内容的显示效果和样式的复用性

功能更强大 例如：给font标签设置超大字体、加粗、斜体

降低了耦合性 css样式专门用于页面的美化、html专门用于页面的搭建

2.2 html与css结合方式

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>02_html与css结合方式</title>
    <!--
        type="text/css"  当浏览器加载此标签 根据type属性类型 解析器切换为 css
    -->
    <style type="text/css">
        span {
            color: blue;
            font-size: 25px;
        }
    </style>

    <!--
        rel="stylesheet"  relation 关系 声明html与引入文件的关系
    -->
    <link rel="stylesheet" href="../css/mycss.css">
    <!--<style>
        @import "../css/mycss.css";
    </style>-->
</head>
<body>
<!--
    html与css结合方式 共有三种
        1）行内样式（一般不用）
            格式：所有标签都有一个共同的属性，在style属性中编写 css样式
        2）内部样式（整个页面）
            格式：使用style标签，在标签体内编写css样式
        3）外部样式（整个网站的使用）
            格式：
                使用link标签，在href属性种引入外部的css文件 (常用)
                使用style标签，在标签体内@import函数引入外部的css文件（加载效率没有link标签高）

    使用优先级：就近原则
            注意：style标签可以出现在页面的任意位置，建议放在head头部 （浏览器加载顺序自上而下）
-->
<span style="color: red;font-size: 20px">java从入门到放弃</span> <br>
<span>php是世界上最美丽的语言</span> <br>
</body>
</html>

1）行内样式
<div style="css样式">    
</div>
2）内部样式
<style>
    css样式
</style>
3）外部样式
<link rel="stylesheet" href="外部css文件路径"/>

2.3 css书写规范

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>03_css书写规范</title>
    <style type="text/css">
        /*
            我是css样式的注释
               格式：选择器{css样式}
                属性名和属性值之间使用冒号分隔
                多个属性之间使用分号分隔
                属性名由多个组成使用减号分隔
                属性值由多个组成使用空格分隔
                属性和属性值不区分大小写，建议小写


        */
        span {
            color: blue;
            font-size: 25px;
            border: 1px solid red;
        }
    </style>
</head>
<body>
<span>php是世界上最美丽的语言</span> <br>
</body>
</html>

2.4 选择器

a）基本选择器

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>04_基本选择器</title>
    <style>
        div {
            color: red;
            font-size: 20px;
        }
        #mydiv{
            color: yellowgreen;
            font-size: 25px;
        }
        .myclass{
            color: skyblue;
            font-size: 30px;
        }
    </style>


</head>
<body>
<!--
    基本选择器 有3个
        1）标签选择器
            格式：标签名{css样式}    div{css样式}
        2）id选择器
            格式：#id属性值{css样式}   #id属性值{css样式}
            特点：id具有唯一性
        3）类选择器
            格式：.class属性值{css样式}   .class属性值{css样式}

   选择器使用优先级：id选择器》类选择器》标签选择器

    注意：给标签设置样式不会指定id，通常使用的class
-->
<div>我是第一种选择器：标签选择器</div>

<div id="mydiv" class="myclass">我是第二种选择器：id选择器</div>


<div class="myclass">我是第三种选择器：类选择器</div>
<p class="myclass">我是第三种选择器：类选择器</p>
</body>
</html>

b）拓展选择器

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>05_拓展选择器</title>
    <style>
        .div2 div .sp1 {
            color: red;
        }

        input[type='text'] {
            background-color: skyblue;
        }

        input[type='text']:focus {
            background-color: lightyellow;
        }

        input[type='password'] {
            background-color: lightpink;
        }

        a:link {
            color: blue;
        }

        a:hover {
            color: red;
        }

        a:active {
            color: blueviolet;
        }

        a:visited {
            color: green;
        }
        font,p,b{
            color: orange;
            font-size: 40px;
        }
    </style>

</head>
<body>
<!--
    拓展选择器
        1）层级选择器
            格式：父选择器 子选择器 孙子选择器...{css样式}
        2）属性选择器
            格式：基本选择器[属性名='属性值']{css样式}
        3）伪类选择器
            同一个标签（元素），在不同操作下呈现出不同的样式
            文本框
                格式：input:focus{css样式}
            超链接
                初始状态 a:link{css样式}
                鼠标移入 a:hover{css样式}
                鼠标点击 a:active{css样式}
                访问过后 a:visited{css样式}
        4）并集选择器
            格式：选择器1，选择器2，选择器3...{css样式}
-->
<!--
    标签.class属性值 tab 创建标签
-->
<div class="div1">
    <div>
        <span class="sp1">span1</span>
        <span class="sp2">span2</span>
    </div>
</div>
<div class="div2">
    <div>
        <span class="sp1">span3</span>
        <span class="sp2">span4</span>
    </div>
</div>
<input type="text"> <br>
<input type="password"> <br>
<a href="#">let me see see</a> <br>

<font>我是字体font</font>
<p>我是段落p</p>
<b>我是粗体b</b>
</body>
</html>

2.5 css常见属性

a）字体和文本

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>06_字体和文本</title>
    <style>
        div{
            font-size: 40px;
            font-weight: bold;
            font-style: italic;
            font-family: 华文琥珀;
            color: red;
            text-align: center;
            line-height: 100px;
            text-decoration: line-through;
        }
        a{
            text-decoration:none;
        }
    </style>
</head>
<body>
<!--
    字体
        font-size：大小
        font-weight：重量
            取值：bold 粗体
       font-style：样式
            取值：italic 斜体
       font-family：类型
            取值：宋体、、、、
    文本
        color：颜色
        text-algin：内容对齐方式
            取值：left 左侧（默认）
        line-height：高度
        text-decoration：装饰
            取值：none 普通文本（默认）、underline 下划线、overline 上划线、line-through 横线（删除线）


-->
<div>城市套路深，我要回农村。。。。</div>
<a href="#">我不需要下划线</a>
</body>
</html>

b）背景属性

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>07_背景属性</title>
    <style>
        body{
/*            background-color: #eaeaea;
            background-image: url("../img/girl3.jpg");
            background-repeat: no-repeat;*/
            background: #eaeaea  url("../img/girl3.jpg") no-repeat;;
        }
    </style>
</head>
<body>
<!--
    背景属性
        background-color:背景色
        background-image:背景图像
            格式：url('图像的文件路径');
        background-repeat：平铺方式
            取值：repeat 水平和垂直（默认）、repeat-x 水平、repeat-y 垂直、no-repeat 不平埔

        进入企业开发，我们一般会简写

        background：背景色 图像 平铺方式

-->
</body>
</html>

c）显示属性

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>08_显示属性</title>
    <style>
        div{
            background-color: orange;
            display: inline;
        }
        span{
            background-color: orange;
            display: block;
        }
        h3{
            display: none;
        }
    </style>
</head>
<body>
<!--
    display:显示属性
        取值：block 块元素、inline 内联元素、none 隐藏
-->
<div>块标签</div>
<span>内联标签</span>
<h3>我是看不到的</h3>
</body>
</html>

d）浮动属性

将标签脱离原有的规则，可以在一行内显示多个

官方的名词：脱离原始文档流

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>09_浮动属性</title>
    <style>
        .box{
            float: left;
            width: 100px;
            height: 100px;
            border: 1px solid red;
        }
    </style>
</head>
<body>
<!--
    float：浮动属性
        取值：left 左浮动、right 右浮动
    clear：清除浮动
        取值：left 清除左浮动、right 清除右浮动、both 清除左右浮动
-->
<div class="box">1</div>
<div class="box">2</div>
<div style="clear: both"></div>
<div class="box">3</div>
<div class="box">4</div>
</body>
</html>

2.6 盒子模型

a）普通盒子

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>10_普通盒子</title>
    <style>
        /*盒子*/
        .box{
            width: 300px;
            height: 300px;
            border: 10px solid red;
            /*padding: 10px 20px 30px 40px;*/ /*上右下左*/
            /*padding: 10px 20px 30px;*/   /*上 左右 下*/
           /* padding: 10px 20px; *//*上下 左右*/
            padding: 10px; /*上下左右*/
            margin: auto;
        }
        /*饼干*/
        .cookie{
            width: 300px;
            height: 300px;
            background-color: darkgoldenrod;
            border-radius: 150px;
        }
    </style>
</head>
<body>
<!--
    普通盒子
        边框
            border-width：宽度
            border-style：样式
                取值：solid 单线、double 双线、dashed 虚线
            border-color：颜色
            简写 border：宽度 样式 颜色

            border-radius：圆角半径 (∟)
        内边距
            padding-top 上
            padding-right 右
            padding-bottom 下
            padding-left 左
            简写 padding：上 右 下 左;
        外边距
            margin-top 上
            margin-right 右
            margin-bottom 下
            margin-left 左
            简写 margin：上 右 下  左；
            特殊：
                margin：auto； 水平对齐
-->
<div class="box">
    <div class="cookie"></div>
</div>
</body>
</html>

b）怪异盒子

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>11_怪异盒子</title>
    <style>
        /*盒子*/
        .box{
            width: 300px;
            height: 300px;
            border: 10px solid red;
            /*padding: 10px 20px 30px 40px;*/ /*上右下左*/
            /*padding: 10px 20px 30px;*/   /*上 左右 下*/
           /* padding: 10px 20px; *//*上下 左右*/
            padding: 10px; /*上下左右*/
            margin: auto;
            box-sizing: border-box;
        }
        /*饼干*/
        .cookie{
            width: 260px;
            height: 260px;
            background-color: darkgoldenrod;
            border-radius: 150px;
        }
    </style>
</head>
<body>
<!--
    普通盒子
        边框
            border-width：宽度
            border-style：样式
                取值：solid 单线、double 双线、dashed 虚线
            border-color：颜色
            简写 border：宽度 样式 颜色

            border-radius：圆角半径 (∟)
        内边距
            padding-top 上
            padding-right 右
            padding-bottom 下
            padding-left 左
            简写 padding：上 右 下 左;
        外边距
            margin-top 上
            margin-right 右
            margin-bottom 下
            margin-left 左
            简写 margin：上 右 下  左；
            特殊：
                margin：auto； 水平对齐

         box-sizing：盒子的类型
            content-box （默认类型）：指定盒子宽度和高度（300*300）之外调整的边框和内边距
            border-box：指定盒子宽度和高度（300*300）之内调整边框和内边距
-->
<div class="box">
    <div class="cookie"></div>
</div>
</body>
</html>
